기본

단축키 : div + 탭버튼
결과값 : <div></div>

클래스 또는 아이디 네임만 입력한 경우

단축키 : .test + 탭버튼
결과값 : <div class="test"></div>

단축키 : #test + 탭버튼
<div id="test"></div>

태그명 입력 없이 클래스명이나 아이디명만 입력후 탭할경우 기본 태그로는 div 를 뿌려준다.

html 코드를 그대로 웹브라우저 화면에 뿌려줘야하는 경우

단축키 : div.test|e
결과값 : <div class="test"></div>

'|' 는 쉬프트키 + \(빽스페이스 좌측의 키) 를 뜻함

심화코드

결과값 : <ul> <li></li> <li></li> <li></li> <li></li> </ul> 단축키 : ul>li.test-$*4
결과값 : <ul> <li class="test-1"></li> <li class="test-2"></li> <li class="test-3"></li> <li class="test-4"></li> </ul> 단축키 : ul>li.test-$@-1*4
결과값 : <ul> <li class="test-4"></li> <li class="test-3"></li> <li class="test-2"></li> <li class="test-1"></li> </ul> 단축키 : p{가는세월}
결과값 : <p>가는세월</p> 단축키 : p{가나다라}>a[href="#" target="_blank"]{마바사}+{아자}
결과값 : <p>가나다라<a href="#" target="_blank">마바사</a>아자</p> 단축키 : header>div{헤더}^footer>div{푸터}
결과값 : <header> <div>헤더</div> </header> <footer> <div>푸터</div> </footer>
